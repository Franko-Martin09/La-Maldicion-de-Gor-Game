# La-Maldicion-de-Gor-Game
La Maldición de Gor:  juego de rol por consola desarrollado en C.
El proyecto fue realizado con fines académicos y pone en práctica
estructuras de datos, lógica de juego y persistencia mediante archivos.

Este proyecto fue realizado en conjunto con compañeros de estudio.
Mi aporte principal incluyó:
- Lógica del juego
- Manejo de estructuras
- Lectura/escritura de archivos
